# Serendipia

## Descrição
Um jogo narrativo pessoal em pixel art feito com JavaScript e p5.js.

## Como jogar
- Setas ou WASD para mover
- E ou Enter para interagir
- Espaço para avançar a carta
- Mouse para clicar nos botões finais

## Como executar
Abrir index.html no navegador ou usar Live Server.

## Música opcional
As músicas do jogo ficam em `assets/music/`:
- `main_theme.mp3` toca do menu até a caminhada no jardim
- `moonwisher.mp3` toca da carta até a tela final

## Observação
Este jogo foi criado como um presente pessoal.
